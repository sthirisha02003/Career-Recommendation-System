# AI Career Advisor Agent

An intelligent AI-based career recommendation system that helps users discover suitable career paths based on their interests, skills, and job descriptions.

## Project Overview

The AI Career Advisor Agent acts like a virtual career counselor. Users can interact with the agent by typing interests such as "data", "finance", or "machine learning", and the system recommends matching careers along with salary range, required experience, and skill suggestions.

## Features

* AI-style conversational interaction
* Career recommendation engine
* Intelligent keyword matching
* Career guidance suggestions
* Salary insights
* Experience recommendations
* Conversation memory
* Fuzzy matching for better career search

## Technologies Used

* Python
* Pandas
* Regex
* Difflib
* Jupyter Notebook

## Dataset Information

The dataset contains:

* Job Title
* Role
* Job Description
* Salary Range
* Experience Required

## Project Workflow

1. Load job description dataset
2. Clean and preprocess text data
3. Create searchable career database
4. Accept user interests
5. Match careers intelligently
6. Provide AI-based recommendations
7. Store conversation memory

## How to Run

1. Clone the repository
2. Open Jupyter Notebook
3. Upload `job_descriptions.csv`
4. Run the notebook
5. Enter career interests
6. Receive AI career suggestions
